Goals for this project:

  1. Create a C# .NET web app that pulls data from the GitHub API about some projects or users that are interesting to you and display that information in a browser. For example, this app should display a list of projects or a list of users.
  2. Create a pass through RESTful service to expose a filtered version of the data as JSON. The service should take a filter parameter and all filtering should be done server side using your code, not using the GitHub API. This service could be a WebAPI or Webservice that returns JSON, searching the GitHub Database and returning a list of results based on the search criteria entered by the user.
